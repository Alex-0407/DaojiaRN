# Daojia-RN
京东到家 React Native 项目。

## 开发环境
```sh
$ npm i -g react-native-cli # 全局安装，只需要安装一次

$ npm install # 安装开发依赖

$ react-native run-ios # mac 环境项目启动

$ npm start # 启动jsBundle
```
## 开发工具

- Nuclide
- Xcode
